Project Name: DigiTools Platforms

Project Description:
Ami "DigiTools Platforms" namer ei project-ti amar web development course "AI Powered Future Ready Web Dev" (Programming Hero, Batch-13)-er Assignment 6 hishebe toiri korechi. Eiti dekhte ekti e-commerce website-er moto holeo, ekhane kono product sell hoy na; borong bivinno dhoroner digital services ekhane rakha hoyeche.

Ei project-e koyekti service card ache ebong protiti card-e ekti button deya ache. Button-e click korle oi service-ti auto-cart-e add hoye jay ebong total price-ta update hoy. User chaile cart theke je-kono service delete-o korte pare, ebong delete korle total price theke oi amount-ti automatic kome jay.

Technologies I used:
Ei project-ti banate ami HTML5, Tailwind CSS, DaisyUI, JavaScript (ES6) ebong React.js bebohar korechi.

Key Features:

Interactive Buttons: Protiti card-er button-e click korle button-er text change hoye jay ebong service-ti cart-e add hoy.

Easy Deletion: Cart section theke je-kono service delete button chepe remove kora jay.

Dynamic Pricing: Service add korle total price bare ebong remove korle price automatic kome jay, ja pura dynamic-ally kaj kore.
